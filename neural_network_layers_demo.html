🔗 Dense (Fully Connected) Layer

layers.Dense(128, activation='relu')

What it does: Every neuron connects to every neuron in the previous layer

Parameters: 128 neurons with ReLU activation

Formula: output = activation(weights × input + bias)

ReLU Activation Function

f(x) = max(0, x)
→
0

🎯 Dropout Layer

layers.Dropout(0.5)

What it does: Randomly sets 50% of neurons to zero during training

Purpose: Prevents overfitting by reducing co-adaptation

Important: Only active during training, disabled during inference

Dropout Effect

During training: Random neurons are "dropped out"

During inference: All neurons are active

📈 Output Layer

layers.Dense(10, activation='softmax')

What it does: Converts outputs to probabilities for each digit (0-9)

Softmax: Ensures all outputs sum to 1.0

Softmax Probabilities

# Complete model architecture model = Sequential([ layers.Flatten(input_shape=(28, 28)), # 784 inputs layers.Dense(128, activation='relu'), # Hidden layer layers.Dropout(0.5), # Regularization layers.Dense(10, activation='softmax') # Output layer ])

Flow: Input (784) → Dense (128) → Dropout → Dense (10) → Probabilities
